# Faster-RCNN-by-C-
implement faster-rcnn in pure C++

1, download caffe-faster-R-CNN from 'https://github.com/ShaoqingRen/caffe', which forked from BLVC/caffe.

2, compile that, and copy the 'libcaffe.so' to this directory.

3, do 'sh build.sh'

3, do './Demo_Detection'
